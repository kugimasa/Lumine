# Lumine

Renderer that renders Luminescence

## Externals

### Libraries

- [DirectX-Headers](https://github.com/microsoft/DirectX-Headers)
- [DirectXTex](https://github.com/microsoft/DirectXTex)
- [DirectXShaderCompiler](https://github.com/microsoft/DirectXShaderCompiler)
- [fpng](https://github.com/richgel999/fpng)
- [imgui](https://github.com/ocornut/imgui)
- [assimp](https://github.com/assimp/assimp)

### Model

- [Stanford Bunny](https://casual-effects.com/data)
    - Place the model under the `Resources/Geo` directory.

### Fluorescence Data

- [Fichet et al. 2024](https://github.com/afichet/fluorescence-rgb)
